Alerta Para Ti

Project information

  • Category: Application Development
  • Client: Atizapan de Zaragoza Municipality
  • Project date: Oct. 2022

Project details

Android application and web page where citizens will be able to access to various information regarding the weather, road accidents, recent earthquakes and medical services stations. The application will also allow the user, in this case admins of the municipality of Atizapan de Zaragoza, to send an alert to the emergency services in case of an emergency.

Developed Competencies

STC0204B: It develops all the designed components of a computer system, based on international standards.

STC0205B: Develop tests that validate compliance with the initial requirements of the computer system.

STC0207B: Manage computing and information technology projects through the efficient use of resources.
